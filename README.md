# Bangladesh Land Measurement Converter
A modern, responsive, and easy-to-use web application designed to convert land area measurements specifically for the Bangladesh context. Built with a focus on accuracy using standard government metrics and a clean UI/UX.

# Features
- Standard BD Metrics: Supports Sqft, Katha, Decimal (Shotok), Bigha, and Acre.

- Bi-directional Conversion: Switch between any two units instantly.

- Modern UI: Built with Tailwind CSS and the Poppins font.

- Responsive Design: Optimized for mobile, tablet, and desktop views.

- Zero Dependencies: Single HTML file with CDN-based assetsâ€”no installation required.

# Convertion Logic Used
The tool uses the following standard conversion factors (based on Bangladesh land measurement standards):

| Unit | Conversion to Sqft | Referece |
| ---- | ------------------ | -------- |
| 1 Decimal (Shotok) | 435.6 sq ft | Standard
| 1 Katha | 720 sq ft | Standard |
| 1 Bigha | 14,400 sq ft | 33 Decimals or 20 Katha |
| 1 Acre | 43,560 sq ft | 100 Decimals |
